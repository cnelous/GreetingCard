# Greeting Card

A simple, interactive birthday greeting card built with HTML and CSS.

## Features
- Clean, modern card design with shadow and rounded corners
- Hover effects (scale + color change)
- Pseudo-elements for decorations (🥳 emojis)
- Interactive links with hover, active, focus, and visited states
- Clickable buttons that show different sections using `:target`
